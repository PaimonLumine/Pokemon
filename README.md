# 宝可梦属性相克表 | Pokémon Type Chart

一个交互式的宝可梦属性相克查询工具，支持单属性和双属性组合，帮助玩家快速了解属性克制关系。

## ✨ 功能特点

- 🎯 **属性组合选择** - 支持选择单属性或双属性组合进行查询
- ⚔️ **进攻端分析** - 显示当前属性对所有其他属性的攻击效果
  - 效果拔群 (x2 伤害)
  - 效果不好 (x0.5 或 x0 伤害)
- 🛡️ **防守端分析** - 显示当前属性组合的防御抗性
  - 弱点 (受到 x2 或 x4 伤害)
  - 抗性 (受到 x0.5 或 x0.25 伤害)
  - 免疫 (受到 x0 伤害)
- 🔧 **特性效果** - 模拟宝可梦特性效果，可手动调整特定属性的伤害倍率
- 🌐 **双语支持** - 支持中文和英文界面切换
- 📱 **响应式设计** - 适配各种屏幕尺寸

## 🎮 使用方法

1. 在浏览器中打开 `type_chart.html` 文件
2. 从属性选择区点击选择第一属性（主属性）
3. 可选择第二属性组成双属性组合
4. 查看下方的进攻和防御分析结果
5. 使用「特性效果」区域模拟特定属性抗性

### 特性效果设置

点击属性右侧的按钮可切换该属性对防御方的伤害倍率：
- **x2** (红色) - 2倍伤害（如受到克制）
- **x1/2** (绿色) - 0.5倍伤害（如抗性）
- **x0** (灰色) - 0倍伤害（如免疫）
